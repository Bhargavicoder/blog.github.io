What is multitasking?

  • Executing several tasks simeltaneously is the concept of multi tasking.
  • There are two types of multitasking
    1. Process Based Multitasking
    2. Thread Based Multitasking
  • Process Based Multitasking:-
Executing several tasks simeltaneously where each task is a seperate independent program is called process based multitasking.
ex:- While typing a java program in the editor we can listen audio songs from same system at the same time we can download a file from internet . All this tasks will be executed simeltaneously and independent of each other. Hence it is process based multitasking .Process based multitasking is best suitable at OS level .
  • Thread Based Multitasking:-
Executing several tasks simeltaneously where each task is a seperate independent task of same program is called thread based multitasking. Application areas of multithreading:-
1)To develop multimedia graphics
2)To devolop animations
3)To devolop video games
4)To devolop webservers and application servers
